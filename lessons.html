Lesson 1: Introduction to the DOM

The Document Object Model (DOM) represents your HTML as a tree of objects. JavaScript can access and manipulate this structure to create interactive websites.

Key Concepts:

  • document.getElementById() selects an element by its ID.
  • textContent or innerHTML modifies the content inside elements.
  • DOM changes reflect immediately on the page without reloading.
// Example:

Hi!


const el = document.getElementById("greet");
el.textContent = "Hello, DOM!";

Change the text of a paragraph with ID demo to "Hello, DOM!"

Lesson 3: JavaScript Functions

      

        
        
      

      

        
Functions are blocks of reusable code that perform specific tasks. They help keep your code clean and organized.

        
function greet(name) {
  return "Hello, " + name;
}
console.log(greet("Emaan")); // Hello, Emaan

        
You can also use arrow functions for shorter syntax:

        
const square = x => x * x;

Lesson 5: Conditional Statements

      

        
        
      

      

        
Use if, else if, and else to control flow based on conditions.

        
const age = 18;
if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

        
switch is useful for comparing exact values:

        
switch (color) {
  case "red": console.log("Stop"); break;
  case "green": console.log("Go"); break;
}

      

      

        
Write code that logs whether a number is positive, negative, or zero.

Lesson 8: Template Literals

      

        
        
      

      

        
Template literals use backticks ` ` and allow variable embedding via ${}.

        
const name = "Emaan";
console.log(`Hi, I'm ${name}`);

        
You can also write multi-line strings:

        
const poem = `Roses are red,
Violets are blue.`;
